# prd-solution-review

Analyze an existing project structure, combine it with a PRD or requirement, and produce a concise review-ready implementation proposal.

## Use When

- You have a PRD and need an engineering plan.
- You need to map requirements to files, modules, APIs, schemas, or tests.
- You want a short review document rather than a long design essay.

## Trigger

```text
$prd-solution-review
```
